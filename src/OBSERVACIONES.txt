Referències

Bibliografia bàsica
Eckel, B. (2009). Piensa en Java. 4a edició. España: Prentice-Hall.
Manual de referència molt popular sobre els aspectes més rellevants de la programació en Java. Especialment útil de cara a fer un repàs de la sintaxi del llenguatge, les seves classes més rellevants i veure els seus aspectes vinculats a l’orientació d’objectes.
Singh, S. (1999). The Code Book: The Science of Secrecy from Ancient Egypt to Quantum Cryptography. U.K: Fourth Estate.
Una revisió de la criptografia, tant des del punt de vista històric com des d’algunes de les bases matemàtiques. Llibre imprescindible de referència tant per aquells lectors que vulguin una visió general del funcionament d’alguns algorismes, com pels que cerquen passar una bona estona amb les seves anècdotes. Absolutament recomanable.
Gong, Li; Ellison, G. (2003). Inside Java 2 Platform Security: Architecture, API Design, and Implementation. 2a edició. USA: Oracle.
Aquest llibre ofereix una visió detallada de l’arquitectura de seguretat de Java, examinant les múltiples amenaces de seguretat existents i estratègies utilitzades per combatre-les. A dins, trobareu una guia pràctica per al desplegament de mecanismes de seguretat a aplicacions Java, així com consells sobre com personalitzar, ampliar i perfeccionar l’arquitectura de seguretat dels vostres programes.
Bibliografia complementària
Oaks, S. (2001). Java Security. USA: O’Reilly.
Llibre orientat a programadors mitjanament experimentats en el llenguatge que repassa l’aplicació d’aspectes lligats a la criptografia a Java. En aquest sentit, cobreix tant la Java Cryptography Architecture (JCA) com la Java Cryptography Extension (JCE), descrivint-ne les classes més significatives.
Long, F; Mohindra, D; Seacord, R.C; Sutherland, D.F; Svoboda, D. (2011). The CERT Oracle Secure Coding Standard for Java. USA: CERT.
Completíssima guia de bones pràctiques de programació segura, enfocades a projectes de tota mena d’envergadura. Proporciona mètriques formals mitjançant les quals avaluar el grau de seguretat d’un codi font.
